# Risetek SYSCAST SERVER DEMO
## 执行方式
* 'mvn exec:java'
* 或者在target目录下：'java -jar risetek.syscast-1.0-SNAPSHOT.jar'
## 输出结果
* 示例程序在局域网络上接收设备发出的UDP广播信号，将接收到的报文转变成字符串在控制台上输出。
